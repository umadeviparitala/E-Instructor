 ---INSTRUCTIONS TO EXECUTE THE PROJECT---

 * Download Visual Studio Code and Nodejs.

 * Open the "E-Instructor" folder in Visual Studio code.

 * Open terminal in Visual Studio code, set path to yogapp with the help of below command

   - cd yogapp [Click Enter]

   and install the below packages by following the commands

   - npm install [Click Enter]
   - npm install cors [Click Enter]
   - npm install express [Click Enter]

   these packages are replicated in package.json file.

 * Use below command to run the project.

   - npm run devstart

 * Enter below link in browser once the project is run.

   - localhost:3000

 * The project runs successfully in your browser.
